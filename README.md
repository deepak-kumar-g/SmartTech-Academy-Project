# SmartTech Academy Project

A simple front-end project for SmartTech Academy containing a landing page and a registration page.

## Description

This repository contains the source files for a small website for SmartTech Academy.
The project includes a main landing page (`index.html`) and a registration page (`register.html`).

## Features

- Simple, responsive HTML pages
- Registration form (static HTML) for collecting user details
- Easy to extend with CSS and JavaScript

## Tech Stack

- HTML

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari)
- (Optional) A simple static server for testing (Python, Node.js, or VS Code Live Server)

## Quick Start

1. Clone the repo:

	 git clone <your-repo-url>

2. Open the site in your browser:

	 - Double-click `index.html`, or
	 - Run a simple HTTP server from the project folder (recommended):

		 - Python 3:

			 ```bash
			 python3 -m http.server 8000
			 ```

		 - Then open `http://localhost:8000` in your browser.

## Project Structure

- `index.html` — Main landing page
- `register.html` — Registration page
- `README.md` — Project documentation

Add any assets (images, styles, scripts) in new folders like `assets/` or `css/` as needed.

## Development

- Edit the HTML files directly for content changes.
- Add CSS files and link them from the HTML to style the pages.
- Add JavaScript to handle form validation or to submit form data to a backend.
